概要

このサイトは けものフレンズ 1話「さばんなちほー」 の再生数、コメント数、マイリスト数を監視しリアルタイムに数値の視覚化を行うことを目的としています。

数値の取得について

ソースとなる数値は ニコニコ動画API を利用し取得しています。そのため、ソース元 API サーバの運用状況により数値の更新が遅延、停止することがあります。

ソース元 API の性質上、意図せず数値が増減することがあります。コメント数とマイリスト数に関してはこの傾向が顕著ですが、異常ではありません。
また、各数値は最大で 1 分程度の遅延を含むことがあります。正確な数値を参照する場合は元動画にアクセスしていただき、ご確認ください。

本サイトではソース元 API を介さず、動画ページから直接的に各数値を取得する方針はありません。

取得期間について

ソース元 API からの取得開始は 2017/09/02 21:30 のため、これ以前のデータの閲覧について以下の制限があります。

  • 各数値の合計値、差分値について1分単位データは取得できず、24時間単位となります。
  • ニコニコチャート 様からのインポートデータとなるため、取得時刻が 5:00 のものとなります。

また動画投稿時間の関係上、2017/01/14 のデータは 17 時間分のデータとなります。

免責

このサイトは個人が運営するファンサイトです。ニコニコ動画、けものフレンズプロジェクトとは一切関係がありません。

このサイトにて公開している数値情報はすべて参考情報であり、正確性の保証しません。取得方法、ソース元APIサーバなどの技術的制約により誤差を含みます。
